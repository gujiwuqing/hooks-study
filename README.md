## useInterval
### Params

| 参数    | 说明                                        | 类型                   |
| ------- | ------------------------------------------- | ---------------------- |
| fn      | 要定时调用的函数                            | `() => void`           |
| delay   | 间隔时间，当取值 `undefined` 时会停止计时器 | `number` | `undefined` |
| options | 配置计时器的行为                            | `Options`              |

### Options

| 参数      | 说明                     | 类型      | 默认值  |
| --------- | ------------------------ | --------- | ------- |
| immediate | 是否在首次渲染时立即执行 | `boolean` | `false` |
